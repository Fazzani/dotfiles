# dotfiles

A repo for storing my personal dotfiles

```bash
# Install all
bash -c "$(curl -fsSL https://raw.githubusercontent.com/Fazzani/dotfiles/master/setup.sh)"

# Without package install
bash <(curl -sSL https://raw.githubusercontent.com/Fazzani/dotfiles/master/setup.sh) --light

# Help
bash <(curl -sSL https://raw.githubusercontent.com/Fazzani/dotfiles/master/setup.sh) --help
```

Into Vim execute :PlugInstall and :PlugUpdate
